> **MissingMiddlewareForInternationalization**: Your configuration setting `i18n.routing: 'manual'` requires you to provide your own i18n `middleware` file.

Votre paramètre de configuration `i18n.routing: 'manual'` exige que vous fournissiez votre propre fichier `middleware` i18n.

## Qu'est-ce qui n'a pas fonctionné ?
Astro génère une erreur si l'utilisateur active le routage manuel, mais n'a pas de fichier middleware.
